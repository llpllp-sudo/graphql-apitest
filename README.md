# SpaceX GraphQL API Testing Suite

This repository contains a comprehensive automated test suite for the **SpaceX GraphQL API**. It utilizes two distinct testing strategies to ensure high reliability and deep functional validation.

## 🚀 Key Features

*   **Hybrid Testing Stack:** Combines the power of **TypeScript/Jest** for unit-like API tests and **Karate DSL** for BDD-style behavioral tests.
*   **GraphQL Specific Validations:** Deep relational query testing, schema validation, mutation authorization, and security checks (depth limiting).
*   **Zero-Install Java Support:** Uses JBang to execute Karate without requiring a pre-installed Global JDK on your machine.
*   **Interactive HTML Reports:** Auto-generated visual reports with full request/response transparency.

---

## 🛠️ Prerequisites

*   [Node.js](https://nodejs.org/) (Version 16 or higher recommended)
*   Internet connection (to pull live SpaceX API data)

---

## 📥 Installation

1. **Clone the repository:**
   ```bash
   git clone <repo-url>
   cd GraphQL-APITest
   ```

2. **Install Node dependencies:**
   ```bash
   npm install
   ```

---

## 🧪 Running Tests

### 1. TypeScript + Jest Tests
Best for strict typing and integration into CI/CD pipelines.
```bash
npm test
```

### 2. Karate BDD Tests
Best for behavioral validation and visual reporting.
```bash
npm run test:karate
```

---

## 📊 Viewing Test Reports

### Karate Reports
After running `npm run test:karate`, an interactive HTML report is generated.
*   **Location:** `target/karate-reports/karate-summary.html`
*   **How to view:** Open the file in any modern web browser to see step-by-step logs, performance metrics, and payload snapshots.

---

## 📁 Project Structure

*   `__tests__/`: TypeScript/Jest functional and security tests.
*   `karate/`: Karate `.feature` files and environment configuration.
*   `src/`: Reusable GraphQL queries, types, and clients.
*   `target/`: Autogenerated test artifacts and HTML reports.
*   `jbang/`: Local JBang runner used for Karate execution (Standalone toolchain).
